Seekerboard Privacy Policy

Effective date: March 20, 2026

Seekerboard is a privacy-first keyboard with optional Solana wallet actions. Normal typing is intended to stay local on-device. Network access is only used when a user explicitly opens wallet features such as connect, receive, send, or Solana Name Service resolution.

What Seekerboard Processes

Seekerboard processes keyboard input locally in order to provide typing, suggestions, emoji, clipboard history, and keyboard customization. If the user opens wallet features, Seekerboard may process Solana public addresses, .sol names, transaction details, and wallet session information needed to complete the requested action.

What Seekerboard Does Not Intend To Do

Seekerboard does not use Google cloud typing services for normal keyboard input. Seekerboard does not sell personal data. Seekerboard is not designed to upload normal typed text to a remote service as part of standard keyboard use.

When Network Access Happens

Network access may occur only when a user explicitly uses wallet features. This can include Solana RPC calls, Mobile Wallet Adapter interactions, and optional Solana Name Service resolution for recipient names.

Clipboard and Wallet Data

Clipboard history, wallet session data, and keyboard preferences may be stored locally on the device to provide app functionality. Users should treat any device-level compromise as a risk to locally stored application data.

Security and User Responsibility

Users are responsible for reviewing wallet approvals, protecting their device, and validating transaction recipients and amounts before signing. Seekerboard is provided with limited warranties to the maximum extent allowed by law.

Contact

For privacy questions, contact the Seekerboard publisher using the support channel listed with the app release.
